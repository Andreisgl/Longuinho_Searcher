Unicode font "Roz Urdu Punjabi Nastaleeq"  Version 2.0
 Posted On October 2012 (Edited on Dec 2, 2021 by admin)  
 ”روز اردو پنجابی نستعلیق“ فونٹ 
Since we established a Punjabi website, the need has been felt to 
  write in Punjabi font, which unfortunately does not fully exist, i.e. 
  to some extent we do have a font but it does not satisfy the needs. We 
  contacted some experts in this field to help and give us necessary 
  guidelines but did not get any encouraging response. Initially we also 
  did not know how to use the software which could enable us to make the font. 
  Work on this project i.e establishing Punjabi’s own font started in 
  September 2009. By the grace of Almighty ALLAH, we can say that we have 
  been able to make our own font in Punjabi (Shahmukhi). If it is widely 
  accepted, we can perhaps use it for other Pakistani languages too in 
  the next version. We request you to give your feedback on it which 
  would enable us to revise it if needed in the next version. Your 
  valuable comments will help us in improving our font's quality.
"Roz Urdu Punjabi Nastaleeq" is developed according to Open Type Font 
  specifications. This font can be used for Urdu and Punjabi languages. 
  It also has extra chracters needed for other Pakistani languages, 
  i.e.
