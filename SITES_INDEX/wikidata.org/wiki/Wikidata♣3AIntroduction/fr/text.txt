Wikidata est une base de données libre, collaborative, multilingue, et secondaire qui collecte des données structurées pour alimenter Wikipédia, Wikimedia Commons, les autres projets du mouvement Wikimédia et pour n'importe qui de par le monde.

Libre. Les données de Wikidata sont publiées sous licence Creative Commons Transfert dans le Domaine Public (CC0 1.0), autorisant ainsi leur réutilisation dans une multitude de cas. Vous pouvez copier, modifier, partager et améliorer les données, même pour une utilisation commerciale, sans avoir à demander la permission.
Collaboratif. Les données sont ajoutées et vérifiées par les contributeurs de Wikidata, qui décident des règles gouvernant la création et la gestion du contenu. Des robots automatisés introduisent aussi des données dans Wikidata.
Multilingue. La modification, la consultation, la navigation et la réutilisation des données sont entièrement multilingues. Les données ajoutées dans une langue sont immédiatement disponibles dans toutes les autres langues. Il est possible et encouragé de contribuer dans plusieurs langues.
Une base de données secondaire. Wikidata enregistre non seulement les déclarations, mais aussi leurs sources et les connexions avec d’autres bases de données. Cela reflète la diversité des connaissances disponibles et soutient le principe de vérifiabilité.
Collecter des données structurées. Le fait d'imposer un haut degré d'organisation structurée facilite la réutilisation des données par les projets Wikimedia et les parties tierces, et permet aux ordinateurs de les traiter et de les « comprendre ».
Assiste les autres projets Wikimedia. Wikidata aide Wikipédia en facilitant la maintenance des boîtes d'informations et des liens vers les autres langues, réduisant ainsi la charge de travail liée aux modifications tout en améliorant la qualité. Les mises à jour faites dans une langue sont rendues disponibles dans les autres langues.
Pour tous. Tout le monde peut utiliser Wikidata de différentes façons et pour n'importe quel usage en utilisant son interface de programmation d'application.
Comment fonctionne Wikidata ?
Ce schéma d'un élément Wikidata montre les termes les plus importants dans Wikidata.
Wikidata est un dépôt central auquel ont accès de nombreux sites, comme les wikis maintenus par la Fondation Wikimedia. Les contenus chargés dynamiquement depuis Wikidata n'ont pas à être maintenus sur chaque projet wiki local. Par exemple, les statistiques, les dates, les lieux et d'autres données communes peuvent être centralisés dans Wikidata.
Le référentiel Wikidata
Les éléments et leurs données sont interconnectés.
Le référentiel Wikidata est constitué principalement d'éléments, chacun ayant un libellé, une description et un certain nombre d'alias. Les éléments sont identifiés de manière unique par un Q suivi d'un nombre, comme Douglas Adams (Q42).
Les déclarations décrivent les caractéristiques détaillées d'un élément et consistent en une propriété et une valeur. Les propriétés dans Wikidata ont un P suivi d'un nombre, comme avec educated at (P69).
Pour un individu, vous pouvez ajouter une propriété pour spécifier où il a été scolarisé, en spécifiant une valeur pour une école. Pour les bâtiments, vous pouvez attribuer des propriétés de coordonnées géographiques en spécifiant des valeurs de longitude et de latitude. Les propriétés peuvent aussi renvoyer vers des bases de données externes. Une propriété qui relie un élément à une base de données externe, telle qu'une base de données d'une autorité de contrôle utilisée par les bibliothèques et les archives, est appelée un identifiant. Les liens de site (sitelinks) spéciaux connectent un élément au contenu correspondant sur les wikis client, comme Wikipedia, Wikibooks or Wikiquote.
Toutes ces informations peuvent être affichées dans n'importe quelle langue, même si les données proviennent de langues différentes. Lors de l'accès à ces valeurs, les wikis client afficheront les données les plus à jour.

Travailler avec Wikidata
Il y a de nombreuses façons d'accéder à Wikidata que ce soit en utilisant des outils intégrés, des outils externes ou des interfaces de programmation.
Wikidata Query et Reasonator sont quelques uns des outils populaires pour rechercher et analyser des éléments Wikidata. La page outils a une longue liste de projets intéressants à découvrir.
Le wikis client peuvent accéder aux données pour leur page en utilisant un interface Lua Scribunto. Vous pouvez récupérer toutes les données indépendamment en utilisant l'API Wikidata.

Les visites guidées Wikidata, conçues pour les nouveaux contributeurs, sont le meilleur endroit pour en apprendre plus sur Wikidata.

Comment puis-je aider ?
Allez-y et contribuez. Contribuer est la meilleure façon d'apprendre la structure et les concepts de Wikidata. Si vous voulez comprendre ces concepts d'entrée de jeu, vous pouvez jeter un œil aux pages d'aide. Si vous avez des questions, n'hésitez pas à les poser sur le bistro du projet ou à contacter l'équipe de développement.
Davantage de sujets à venir
Wikidata est un projet vivant avec un développement actif. Dans le futur, il y aura davantage de types de données et d'extensions disponibles. Vous trouverez plus information sur Wikidata et son développement en cours sur la page Wikidata à Meta. Si voulez recevoir une information actualisée concernant le développement, et participer à des débats sur le futur du projet, inscrivez-vous à la liste de diffusion de Wikidata.
