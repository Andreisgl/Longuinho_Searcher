This is a list of all current Wikimedia Foundation projects. Please note that this is not a list of sites using the MediaWiki software.
There is an overview of large Wikimedia projects and a List of Wikipedias.
For a tabular view of projects, see Table of Wikimedia projects (by size), or the automatically updated site matrix.
For proposed projects, see category of proposed projects and Proposals for new projects.
To download a copy of a project, see dumps.wikimedia.org.
For a complete list of mailing lists, IRC and other channels, see List of communications channels.
