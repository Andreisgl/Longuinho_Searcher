This user account is a php-based bot.
It is not a sock puppet, but rather an automated account for making repetitive edits that would be extremely tedious to do manually.
List users with ssh access to CommonsDelinker
tasks • contribs • count • logs • page moves • block user • block log • flag log  • global contribs • flag bot
This bot runs on Wikimedia Toolforge.
Administrators: If this bot needs to be blocked due to a malfunction, please remember to disable autoblocks so that other Toolforge bots are not affected.
See m:User:CommonsDelinker for the multi project tasks that CommonsDelinker performs.
Search engine for CommonsDelinker edits
Here on Wikimedia Commons, CommonsDelinker has the following tasks:
Delinking deleted images
Replacing images
Add commands at User:CommonsDelinker/commands and make requests at User talk:CommonsDelinker/commands. 
CommonsDelinker written by Orgullomoore and Bryan.  In January 2015 delinker has been rewritten by Magnus Manske.
