Kashamarka suyu
Kashamarka suyu (Kashamarka suyu nishqapis), (Kastilla shimichaw: Departamento de Cajamarca) nishqaqa huk piruwanu suyum, piqa markanqa Kashamarka markam. 2017 watachaw, kay suyuchaw 1 383 519 runam tiyarqayan.

Hispanohablantes
Hola y bienvenido a esta nueva wikipedia en quechua norperuano! Si hablas poco o nada de este idioma, también puedes ayudar, por ejemplo poniendo imágenes en los artículos que ya existen, puedes informarte más en:Wikipidiya:Hispanohablantes. Para cualquier duda o pregunta, o si quieres ayudar y no sabes cómo, puedes hacer tu pregunta en la Chichería. Si deseas mayor información de este proyecto puedes escribir a: user:Katariq o katariq@hotmail.com.
English speakers
Hello and welcome to this wikipedia in Inkawasi-Cañaris-Cajamarca Quechua. If you would like to help the Quechua Wikipedia become a great success but you do not speak any Quechua, there is still a lot you can do, for example adding images to existing articles. Read more. If you want more information about this project, you can write to: user:Katariq or to katariq@hotmail.com.
