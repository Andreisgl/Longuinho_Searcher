Plusieurs raisons peuvent expliquer cela : peut-être l’avez-vous déjà utilisé (il n’est valable qu’une fois). Il peut aussi ne pas être compatible avec l’article que vous avez choisi, ou n’être valable la page d'accueil Mozilla. qu’avec un minimum d'achat. Souvent, un code promo n’est pas cumulable avec une autre réduction, ce qui peut expliquer qu’il ne fonctionne pas. Enfin, il peut aussi être arrivé à expiration, car les coupons de réduction ont une durée limitée.        🛍 Puis-je utiliser un bon d'achat expiré ? En général, non, car le bon de réduction fonctionne avec une date de fin, à partir de laquelle il n’est plus utilisable. Mais il arrive que certains code pourtant expirés fonctionnent encore, c’est pour cela que nous continuons à les lister quelques jours après leur date d’expiration.    🏷 Code promo avec une remise : qu’est ce que c’est ? Il s’agit d’un bon de réduction avec un rabais qui vous permet d’économiser le montant indiqué sur le coupon, directement lors de vos achats en ligne. Un minimum de commande est souvent requis. ⁒ Code promo avec une remise en pourcentage : qu’est ce que c’est ? C’est le bon de réduction «  classique » qui vous permet d’économiser le montant indiqué sur le coupon de réduction, en fonction du total de votre commande. Un minimum de commande ou des limitations peuvent s’appliquer et dans ce cas, sont précisées dans les conditions de vente.    🎁 Livraison gratuite à domicile : qu’est ce que c’est ?C’est une réduction qu’on apprécie particulièrement et qui permet de recevoir sa commande sans frais supplémentaires. C’est particulièrement agréable dans le cas de commandes d’objets volumineux ou encombrants, comme un canapé, un écran plat ou vos courses en ligne.
