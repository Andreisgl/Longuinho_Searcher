Hispanohablantes
Hola y bienvenido a esta nueva wikipedia en Quechua de Yauyos! Si hablas poco o nada de este idioma, también puedes ayudar, por ejemplo poniendo imágenes en los artículos que ya existen, puedes informarte más en:Wikipidiya:Hispanohablantes. Para cualquier duda o pregunta, o si quieres ayudar y no sabes cómo, puedes hacer tu pregunta en la Chichería. Si deseas mayor información de este proyecto puedes escribir a: user:Michael Obregón (o a michaeljuniorobregonpozo@gmail.com), y user:Katariq (o katariq@hotmail.com).
English speakers
Hello and welcome to this wikipedia in Yauyos Quechua. If you would like to help the Quechua Wikipedia become a great success but you do not speak any Quechua, there is still a lot you can do, for example adding images to existing articles. Read more. If you want more information about this project, you can write to: user:Katariq or to ted.pari@unmsm.edu.pe
