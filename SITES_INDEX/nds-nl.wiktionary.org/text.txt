SIL/ISO 639-3: nds-nlEthnologue: nds-nlGlottolog: nds-nl
 This project uses a different ISO code.
This code corresponds to an existing Wikimedia project, which can be found at wikt:nds:. 
Although there are separate projects for Dutch Low Saxon Wikipedia at nds-nl: and Low German Wikipedia at nds:, Dutch Low Saxon has been incorporated into the Low German Wiktionary; see wikt:nds:Wiktionary:Veurblad. Please contribute there.
You can search for other projects in this language: Wikipedia · Wiktionary · Wikibooks · Wikinews · Wikiquote · Wikivoyage · Wikisource · Wikiversity
